# Fintech User Retention & Cohort Analysis

## Overview
Analyzed fintech customer retention trends, cohort behavior, and revenue activity using SQL and Python.

## Tools Used
- SQL
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Cohort Retention Analysis
- Revenue Trend Analysis
- User Engagement Tracking
- Business KPI Reporting

## Key Insights
- Early signup cohorts demonstrated stronger retention performance
- Revenue trends aligned closely with transaction activity
- Cohort analysis identified user engagement decline across later cohorts
